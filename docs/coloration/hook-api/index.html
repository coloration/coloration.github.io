# HookStore[class]

简易的状态管理 Hooks. 没有设计 Action 层。如果有需要可以使用 useStore 进行下一层封装，或者继承 ReactStore 进行拓展，实例内部只是缓存了 store<T> , listeners , mapCache .

不像之后的 hooks 具有唯一性，hookStore 以类的方式实现，以增加了复用性.
