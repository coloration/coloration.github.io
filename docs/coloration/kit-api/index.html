# curry

柯里化一个函数，将参数缓存起来，返回一个新的函数。只接受剩下的参数。如果再次传入的参数不足，则继续进行柯里化.

Note: 也是为了配合这个特性， kit 中所有的函数的配置函数都靠前，数据参数都靠后 (与 ramda 相同，与 lodash 不同). 这符合函数式编程的习惯.

# equal

判断两值相等，如果没有指定判断函数则使用 Object.is 判断.
