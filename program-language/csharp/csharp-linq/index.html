# IEnumerable 和 IEnumerable<T>

我们只能对实现了 IEnumerable 或 IEnumerable<T> 的接口的对象进行 Linq;
因为他们实现 GetEnumerator {IEnumerator} 方法。换句话说我们只能对可枚举的对象进行 Linq。
